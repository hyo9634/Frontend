웹브라우저는 '미디어 쿼리' 기능을 지원

-미디어 쿼리 : 미디어 타입에 따라 CSS를 다르게 적용하는 기능
                프린터, 데스크탑, 스마트폰, 태블릿, 키오스크, 시각장애용 웹 브라우저 마다 적합한 웹페이지를 구현할 수 있게 지원한다.

결론
다양한 방치에 따라 해상도 크기 및 비율이 제각각 다른 경우에
미디어 쿼리를 적용한 웹페이지는 개발 효율성과 유지보수에 큰 장점이다.
반응형 웹 페이지는 미디어 쿼리를 사용하여 개발한다.

2. 설정
    2-1. meta tag : 웹 페이지에 추가 정보를 제공
    <mata name="title" content="신세계 INC 자바 개발자 미니프로젝트">
    <meta name="description" content="신세계 INC 프로젝트 포트폴리오"> --검색엔진 노출

    2-2. viewport = 핸드폰 버전 등의 화면을 볼 때 필요함
        <meta name = "viewport" content ="width height initial-scale = 2.0 (초기확대 비율)"> --화면의 비율을 설정

    2-3. 미디어쿼리
    @media (<미디어쿼리){
        CSS코드
    }

    <link rel="stylesheet" href="파일이름" media="미디어쿼리"> 적용

@-규칙 : CSS 내부에서 특정한 규칙을 표현하는 방식
