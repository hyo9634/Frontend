표현식 : 값을 만들어 내는 간단한 코드
문장 : 프로그래밍 언어에서 실행할 수 있는 코드의 최소 단위, ; 문장을 끝냄.
ㄴ>표현식이 모여 문장을 만들고 문장이 모여 프로그램을 만듦
키워드 : 예약어 자바스크립트를 처음 만들때 정해진 특별한 의미가 부여된 단어
        break, else, if, instance of, true, case, false, try...
식별자 : 자바스크립트에서 변수나 함수 등 이름에 붙일 때 사용하는 단어
            키워드 사용 금지, 특수문자 _와 $ 허용, 공백 입력X (has_Space)...
            (1) 클래스 이외의 생성자 함수 이름은 항상 대문자로 시작
            (2) 변수, 인스턴스 함수, 매서드는 항상 소문자 시작
            (3) 여러개의 단어로 된 식별자는 첫글자 대문자 
주석 : ctrl + / //한 줄 주석, /* 여러 줄 주석 */

자바스크립트 출력 : 기본 출력 방법 alter("메세지")브라우저 출력, CLI => console.log ()
